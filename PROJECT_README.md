# LabMtuci - Система сдачи лабораторных работ

Полнофункциональная веб-система для управления лабораторными работами студентов с поддержкой оценивания преподавателями.

## 🎯 Основные возможности

### Для студентов
- ✅ Аутентификация с логином/паролем
- ✅ Просмотр 5 лабораторных работ
- ✅ Загрузка файлов работ через S3
- ✅ Просмотр оценок и отзывов преподавателя
- ✅ Отслеживание статуса (не отправлено/отправлено/оценено)

### Для преподавателей
- ✅ Аутентификация с логином/паролем
- ✅ Просмотр всех студентов и их работ
- ✅ Оценивание работ (0-100)
- ✅ Добавление отзывов к работам
- ✅ Загрузка файлов студентов
- ✅ Фильтрация по лабораторным работам

## 🛠️ Технологический стек

### Backend
- **Express.js** - веб-сервер
- **tRPC** - типобезопасный RPC фреймворк
- **PostgreSQL/MySQL** - база данных
- **Drizzle ORM** - ORM для работы с БД
- **JWT** - аутентификация на основе токенов
- **bcrypt** - хеширование паролей
- **S3** - хранилище файлов

### Frontend
- **React 19** - UI библиотека
- **Tailwind CSS 4** - утилиты для стилизации
- **Wouter** - маршрутизация
- **shadcn/ui** - компоненты UI
- **Sonner** - уведомления

## 🗄️ Схема БД

### users
- id (PK)
- username (UNIQUE)
- email
- passwordHash
- name
- role (student | teacher)
- createdAt, updatedAt

### laboratories
- id (PK)
- number (1-5)
- title
- description
- topic
- createdAt, updatedAt

### submissions
- id (PK)
- studentId (FK)
- labId (FK)
- fileUrl
- fileKey (S3)
- fileName
- status (not_submitted | submitted | graded)
- grade (0-100)
- feedback
- submittedAt, gradedAt
- createdAt, updatedAt

## 🔌 API Эндпоинты (tRPC)

### Аутентификация
- `auth.login` - Вход в систему
- `auth.logout` - Выход из системы
- `auth.me` - Получить текущего пользователя

### Лабораторные работы
- `labs.list` - Получить все лабораторные работы

### Студент
- `student.submissions` - Получить все работы студента
- `student.getSubmission` - Получить конкретную работу
- `student.uploadFile` - Загрузить файл работы

### Преподаватель
- `teacher.students` - Получить всех студентов
- `teacher.submissions` - Получить работы по лабораторной
- `teacher.gradeSubmission` - Оценить работу
- `teacher.downloadFile` - Загрузить файл

## 📝 Примечания

- Все пароли хешируются с использованием bcrypt
- JWT токены действуют 7 дней
- Файлы хранятся в S3 с генерируемыми ключами
- API защищена middleware аутентификации
- Роли разделены на студента и преподавателя
